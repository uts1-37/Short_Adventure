# Short Adventure

A 2D platformer game built with the Godot Engine.
Guide your character through a short adventure — run, jump, and explore the stage while avoiding obstacles.

<img width="1154" height="680" alt="gameplay" src="https://github.com/user-attachments/assets/23157472-5b8d-4348-b28c-b1551be71a54" />

## Download & Play

1. Download **`Jumpgame.zip`** from this repository.
2. Extract the zip file.
3. Run the executable to start the game.

## Controls

| Action | Key |
|--------|-----|
| Move Left | `←` or `A` |
| Move Right | `→` or `D` |
| Jump | `Spacebar` |

## Built With

- **Engine**: Godot Engine
- **Genre**: 2D Platformer
